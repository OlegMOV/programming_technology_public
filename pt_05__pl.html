1. Розробка програми з циклом по колекції

Теорія - 5 хв, практика - 15 хв

🎯Колекція - це послідовність змінних одного чи більшої кількості типів, яка поєднується в одній змінній.

Рис. 1.1

Коли у заголовку циклу реалізується звернення до колекції, то на кожній ітерації видається посилання наступну змінну з цієї колекції.

Порядок елементів у списку (індекс) починається з нуля

🧠Практичне завдання 1

На озброєнні у підрозділі Браво є кілька типів БпЛА.

  1. Розробити програму, яка дозволить уточнити кількість типів БпЛА наявних у підрозділі, а у подальшому забезпечить послідовне введення назви типу БпЛА оператором.
  2. Для збереження введених даних використати структуру даних - список (list).

🧠Практичне завдання 2

Вважати, що список БпЛА, отриманий у першому завданні, введено у порядку їх надходження у підрозділ.
Розробити програму, що дозволить:

  1. Вивести список назв типів БпЛА від новішого до найстарішого.
  2. Реалізувати нумерацію списку.

Висновки:
2. Циклічний оператор по колекції дозволяє скоротити код програми при збереженні необхідної логіки програми.

У програмуванні цей метод називають LIFO (англ. Last In, First Out). Є також зворотній метод FIFO (англ. First In, First Out)
Український переклад FIFO — принцип «чотирьох П» (Першим Прийшло — Першим Пішло)

3. Розробка програми з умовою продовження

Теорія - 5 хв, практика - 15 хв

Розглянутий циклічний оператор for по суті є покращенням лінійної програми. Яка реалізує послідовне виконання інструкцій коду програми. Але оскільки програми розробляють для обробки даних, які можуть мати різні значення є необхідність мати можливість реагувати на зміну діапазонів цих значень. Наприклад, у автомобілі програма бортового комп'ютера видає сигнал попередження при зміні температури менше 5 °С щодо можливості обмерзання дороги.
При розробці програм доволі часто виникає питання отримання певної вибірки значень для перевірки дієздатності розробленого коду програми. Для цього є вбудована бібліотека random, яка має ряд функцій, що дозволяють отримати псевдовипадкові дані.

Як для будь-якої іншої функції/ класу / модуля / бібліотеки мови програмування Python, інформацію про неї можна отримати в командному рядку інтерпретатора:
>>> import random
>>> help(random)

Застосуємо дану бібліотеку для отримання випадкових значень температури зовнішнього середовища. Для цього використаємо функцію random.randrange(start, stop[, step]), де - start, stop початкове та кінцеве значення числового інтервалу для вибору значень, а step - крок значень.

import random
random_value = random.randrange(-19,75,1)

У зазначеному прикладі здійснюється підключення бібліотеки та отримується випадкове значення в інтервалі [-19; 75].

🧠Практичне завдання 4

Розробити код програми, що дозволяє постійно стежитиме за температурою зовнішнього середовища, та у випадку зниження температури видавати повідомлення у консоль наступного виду:

|------------------------------|
| Температура ззовні: -3 °С |
| Обережно, можлива ожеледиця! |
|------------------------------|
Для реалізації безконечного циклу можна застосувати циклічний оператор while із зафіксованою умовою продовження
while True:
    <Тіло циклу>
Для завершення безконечного циклу програми у консолі інтерпретатора необхідно натиснути наступні гарячі клавіші ctrl+c
При редагуванні коду програми важливо застосовувати гарячі клавіші, що забезпечить підвищення швидкості написання програми.
Для копіювання рядку коду та його переміщення:
  1. Поставити курсор у будь-якому місці рядку, який потрібно скопіювати
  2. Натиснути гарячі клавіші ctrl+c - що дозволить скопіювати рядок
  3. Натиснути гарячі клавіші ctrl+v - що дозволить вставити даний рядок знизу
  4. Перевести курсор на скопійований рядок.
  5. Затиснувши клавішу alt стрілками вверх / вниз перемістити скопійовану строку на необхідне місце у коді.
