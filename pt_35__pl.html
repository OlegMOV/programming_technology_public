1. Practical Tasks: Python Classes

Task 1: Basic Class Creation

Description:
Create a class Car with attributes brand and year. Write a method display_info() that prints the car’s brand and year.
Expected result:

car = Car("Toyota", 2020)
car.display_info()  # Brand: Toyota, Year: 2020

Task 2: Adding Methods

Description:
Extend the Car class with a method start_engine() that prints "The car is starting...".
Expected result:

car.start_engine()  # The car is starting...

Task 3: Constructor and Default Values

Description:
Create a class Student with attributes name and grade. The grade should default to "Not assigned". Add a method set_grade() to update the grade.
Expected result:

s = Student("Alice")
print(s.grade)  # Not assigned
s.set_grade("A")
print(s.grade)  # A

Task 4: Using __str__

Description:
Modify the Student class to include a __str__ method that returns a string like "Student Alice, Grade: A".
Expected result:

print(s)  # Student Alice, Grade: A

Task 5: Inheritance

Description:
Create a base class Animal with a method speak(). Then create subclasses Dog and Cat that override speak() to print "Woof!" and "Meow!".
Expected result:

dog = Dog()
cat = Cat()
dog.speak()  # Woof!
cat.speak()  # Meow!

Task 6: Class Variables

Description:
Create a class Book with a class variable library_name = "City Library". Each book has attributes title and author. Add a method info() that prints the book details along with the library name.
Expected result:

b = Book("1984", "George Orwell")
b.info()  # Title: 1984, Author: George Orwell, Library: City Library

Task 7: Combining Classes

Description:
Create a class BankAccount with attributes owner and balance. Add methods deposit(amount) and withdraw(amount) that update the balance. Then create a class Bank that stores multiple accounts in a list and has a method show_accounts() to display all owners and balances.
Expected result:

acc1 = BankAccount("Alice", 1000)
acc2 = BankAccount("Bob", 500)
bank = Bank([acc1, acc2])
bank.show_accounts()
# Alice: 1000
# Bob: 500

✨ These tasks gradually introduce constructors, methods, __str__, inheritance, class variables, and aggregation

2. Practical Tasks: File Operations with Class Methods

Task 1: Writing to a File

Description:
Create a class Logger with a method write_log(message) that writes the message into a file log.txt. Each call should append a new line.
Expected result:

logger = Logger()
logger.write_log("Program started")
logger.write_log("Error: invalid input")
# log.txt → 
# Program started
# Error: invalid input

Task 2: Reading from a File

Description:
Extend the Logger class with a method read_logs() that reads all lines from log.txt and prints them.
Expected result:

logger.read_logs()
# Output:
# Program started
# Error: invalid input

Task 3: Saving and Loading Objects

Description:
Create a class Student with attributes name and grade. Add methods save_to_file(filename) and load_from_file(filename) to store and retrieve student data in text format.
Expected result:

s = Student("Alice", "A")
s.save_to_file("student.txt")

s2 = Student("", "")
s2.load_from_file("student.txt")
print(s2.name, s2.grade)  # Alice A
