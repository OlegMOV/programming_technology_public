1. Редактори початкового коду. Огляд відомих рішень

Теорія - 10 хв, практика - 0 хв

Редактор початкового коду - комп'ютерна програма, що забезпечує написання та редагування тексту програми. По суті своїй це текстовий редактор. Як правило для розробки комп'ютерних програм застосовуються інтегровані середовища розробки. Інтегроване середовище розробки - комплексне програмне рішення, яке містить у собі редактор початкового коду та доповнення, що забезпечують автоматизацію написання та відлагодження коду програми. Можуть містити у собі компілятор / інтерпретатор. Як правило це платні програмні продукти.

Одним із популярних редакторів початкового коду є Visual Studio Code або VS Code. Це комп'ютерна програма розроблена корпорацією  Microsoft, яка може функціонувати на базі поширених операційних систем: Windows, Linux, MacOS. До основних складових редактора можна віднести:

Рис. 1.1

Підсвітка синтаксису - дозволяє виділяти різним кольором / написанням інструкції мови програмування (службові команди), змінні та значення змінних, коментарі, синтаксичні помилки.  

Автодоповнення - (IntelliSense - технологія автодоповнення Microsoft) дозволяє дописувати інструкції обраної мови програмування та інші сутності у коді програми по їхнім початковим символам у залежності від контексту.  

Відлагодження - процес пошуку помилок у початковому коді програми.      

Контроль змін або Система керування версіями файлів - комп'ютерна програма, що стежить за змінами у файлах та їх документує. Тим самим забезпечуючи можливість відмінити зміни, а також організувати роботу багатьох розробників в одному проєкті.  

Редактор VS Code на сьогоднішній день є одним із найпопулярніших безкоштовних редакторів, який застосовується розробниками програмного коду майже на усіх відомих мовах програмування. За рахунок доповнень, які можливі вільно встановлювати у редактор, є можливість автоматизувати широкий спектр загальних та специфічних завдань у процесі розробки. А також налаштувати положення панелей інструментів, їх кольорову палітру. Для доповнень є виділений електронний ресурс де їх можна завантажити та встановити. Для цього також є відповідна панель інструментів у редакторі.

Даний редактор початкового коду має доволі велику спільноту, яка працює у даному редакторі, розробляє розширення та доповнення до нього. Що забезпечує можливість швидких оновлень та врахування усіх потреб для забезпечення ефективного написання коду.

Тому редактор VS Code у сукупності з розробленими доповненнями, може, і це нижче буде продемонстровано, бути налаштовано на комфортне і продуктивне середовище для розробки комп'ютерних програм мовою Python.

Одним із відомих інтегрованих середовищ для розробки програм мовою Python є проєкт PyCharm від чеської компанії JetBrains. Він являє собою комплексне рішення, яке містить у собі усі наведені на рис. 1.1 компоненти, і готовий до застосування "із коробки". Але більш широкий функціонал доступний лише у платних версіях.

4. Техніка розробки програми

Теорія - 5 хв, практика - 10 хв

Як правило, для розробки комп'ютерної програми створюється окремий проєкт. По суті це окрема тека (папка / директорія) де розміщуються усі файли, необхідні для розробки:

  1. Файли програми програмного коду.
  2. Теки для розміщення медіа файлів (картинки / відео), окремих модулів та ін.
  3. Службові файли з налаштуваннями допоміжних систем для забезпечення ефективної розробки.

Зазначена тека може бути створена у провіднику, за допомогою команд у операційній системі, яку використовуєте, або безпосередньо у редакторі. Для цього у вікні вітання обирається пункт відкрити теку (Open Folder)
