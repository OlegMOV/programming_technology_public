НАВЧАЛЬНІ ПИТАННЯ

1. Списки. Кортежі

Теорія - 30 хв, практика - 0 хв

Як правило, за напрямом подальшої службової діяльності, комп'ютерні програми застосовуються для автоматизації обробки масивів даних. Наприклад, текстові повідомлення із месенджерів або електронних джерел, відомості щодо чисельності та стану підрозділів, опис та характеристика об'єктів. У ході обробки таких даних отримують додаткові відомості, які дозволяють у подальшому проводити аналіз та приймати рішення на його основі.
Для впорядкування таких відомостей використовують структури даних - спосіб організації даних.
У мові Python існують вбудовані структури даних, серед яких є: послідовності (списки, кортежі, діапазони), бінарні послідовності, рядки, множини, словники.

Список - це тип даних, де міститься від 0 і більше значень у порядку їх запису під час його створення. Кожне окреме значення зі списку називають елементом списку. Ці значення можуть бути різних типів. Записуються у квадратних дужках через кому.

programming_languages = ['Python', 'C++', 
     'Javascript', 'Go', 'Java']

Для задання порожнього списку можна скористатися двома варіантами

# Варіант 1
empty_list = []
# Варіант 2
empty_list = list()

1.1 Застосування індексів елементів списку

Доступ до елементів списку можливий за допомогою індексів - порядкового номеру знаходження їх у послідовності.

Індекси починаються з нуля. Тобто, перший у списку елемент має індекс 0.

Для доступу до конкретного елементу використовують змінну і квадратні дужки з індексом.

programming_languages = ['Python', 'C++', 
     'Javascript', 'Go', 'Java']
programming_languages[0] # 'Python'
programming_languages[2] # 'Javascript'

Індекси можуть бути від’ємними, у такому випадку нумерація буде відбуватися з кінця (кількість елементів списку + від’ємний індекс)

programming_languages = ['Python', 'C++', 
      'Javascript', 'Go', 'Java']
programming_languages[-1] # 'Java'
programming_languages[-2] # 'Go'

Використовуючи індекси можливо отримувати доступ до частини елементів списку.

Таку операцію називають зріз - отримання зі списку частини його елементів. Синтаксис такої операції наступний name_variable_list[index_start: index_end] або name_variable_list[index_start: index_end: step]

programming_languages = ['Python', 'C++', 
        'Javascript', 'Go', 'Java', 'Perl']
programming_languages[2:4] # ['Javascript', 'Go']
# Вибір кожного другого елементу списку 
programming_languages[0:4:2] # ['Python', 'Javascript']

Можливо також застосовувати від'ємні індекси 

programming_languages = ['Python', 'C++', 
            'Javascript', 'Go', 'Java', 'Perl']
programming_languages[2:-1] # ['Javascript', 'Go', 'Java', 'Perl']
	```

>[!TIP] Зручно використовувати для позначення зрізу, що починається від конкретного значення і до кінця списку, у випадку коли кількість елементів списку є змінною величиною

За результатами операції зрізу створюється новий список, якому можна призначити змінну і використовувати у коді програми. Крім операції зрізу, для створення нового списку застосовують операцію об'єднання списків
```python
programming_languages_1 = ['Python', 'Javascript']
programming_languages_2 = ['C++', 'Go', 'Java', 'Perl']
programming_languages_union = programming_languages_1 \
    + programming_languages_2
print(programming_languages_union)
# ['Python', 'Javascript', 'C++', 'Go', 'Java', 'Perl']

Для створення списку з однаковими значеннями застосовують наступний синтаксис

start_values = [0]*7 # >>> [0,0,0,0,0,0,0]

1.2 Редагування елементів списку

Список є змінюваним типом даних, отже елементи списку можна редагувати - змінювати та видаляти.
Для зміни елементів застосовують індекси

programming_languages = ['Python', 'C++', 
        'Javascript', 'Go', 'Java', 'Perl']
# Для заміни одного елементу
programming_languages[3] = 'Pascal'  
# ['Python', 'C++', 'Javascript', 'Pascal', 'Java', 'Perl']
# Для зміни кількох елементів в одній операції
programming_languages[1:3] = ['Pascal', 'PHP'] 
# ['Python', 'Pascal', 'PHP', 'Pascal', 'Java', 'Perl']
При редагуванні списку через зріз інтервал зрізу і кількість елементів, що вставляються можуть відрізнятися

Для видалення елементів масиву застосовують оператор del(<index>) або del(<[start_index: end_index]>). Синтаксис такої операції наступний

programming_languages = ['Python', 'C++',   
        'Javascript', 'Go', 'Java', 'Perl']
# Для видалення одного елементу
del programming_languages[1]
# ['Python', 'Javascript', 'Go', 'Java', 'Perl']
# Для видалення кількох елементів в одній операції
del programming_languages[1:3]
# ['Python', 'Go', 'Java', 'Perl']

1.3 Методи списку

Для проведення інших операцій над списками є ряд методів
Для додавання та видалення елементів:

Метод Приклад застосування
list.append(x) numbers = [1, 2]; numbers.append(3);
# [1,2,3]
list.extend(iterable) numbers = [1, 2]; numbers.extend([3,4]);
# [1,2,3,4]
list.insert(n, x) numbers = [1, 2]; numbers.insert(1, 7);
# [1,7,2]
list.remove(x) numbers = [1, 2, 3]; numbers.remove(1);
# [1,3]
list.pop(x) numbers = [1, 2, 3]; new_list = numbers.pop(1);
# numbers = [1,3]; new_list = 2
list.pop() numbers = [1, 2, 3]; new_list = numbers.pop();
# numbers = [1,2]; new_list = 3
list.clear() numbers = [1, 2, 3]; numbers.clear();
# numbers = []

Для пошуку та сортування елементів списку:

Метод Приклад застосування
list.index(x) numbers = [1, 2]; numbers.index(2);
# 1
list.index(x) numbers = [1, 2]; numbers.index(3);
# ValueError
list.count(x) numbers = [1, 21, 5, 21]; numbers.count(21);
# 2
list.sort(key=None, reverse=False) numbers = [1, 21, 5, 21]; numbers.sort();
# [1, 5, 21, 21]
list.sort(key=None, reverse=False) numbers = [1, 21, 5, 21]; numbers.sort(reverse=True);
# [21, 21, 5, 1]
list.reverse() numbers = [1, 21, 5, 21]; numbers.reverse();
# [21, 5, 21, 1]
  • list.index(x) - повертає індекс першого входження елемента x у зрізі list[start: end] (необов'язкові параметри start та end інтерпретуються як нотації зрізу). Значення, що повертається, є індексом списку list. Якщо елемента в списку не знайдено, виникає виняток ValueError;
  • list.count(x) - повертає кількість входжень елемента x в список;
  • list.sort(key=None, reverse=False) - відсортовує елементи списку (аргументи методу можуть бути використані для налаштування сортування). За замовчуванням сортування відбувається за зростанням. Для сортування в зворотному порядку використовуйте параметр
    reverse = True. У результаті сортування змінюється сам список.
  • list.reverse() - змінює порядок розташування елементів у списку на зворотний. Змінюється сам список.
  • list.copy() - повертає копію списку. Аналогічно a[:].
    До списку також застосовують вбудовані функції інтерпретатора Python:
  • print(list_variable) - виводить список у консоль;
  • len(list_variable) - повертає

1.4 Вкладені списки

Враховуючи те, що елементами списку можуть бути змінні різних типів, у тому числі і списком. При цьому така структура даних називається вкладені списки. Для прикладу, при впорядкуванні даних списку осіб, одним із елементів може бути номер телефону. Оскільки номерів може бути декілька, тому доцільно використати список

persons = [
    ['person_1', ['(xxx) xxx-xx-xx', '(yyy) yyy-yy-yy']], 
    ['person_2', ['(xxx) xxx-xx-xx', '(yyy) yyy-yy-yy']]]
Запис інструкції у кілька рядків можливий у випадку використання дужок

У вищенаведеному прикладі використано подвійну вкладеність - загальний список, у нього вкладений список осіб у якому є елемент список телефонів особи.
Доступ до елементів вкладених списків здійснюється за допомогою індексів починаючи від зовнішнього до внутрішніх, наприклад

complex_list = ['first', ['second_1', 'second_2']]
complex_list[0] # 'first'
complex_list[1] # ['second_1', 'second_2']
complex_list[1][0] # 'second_1'
complex_list[1][-1] # 'second_2'

1.5 Кортеж

Схожою але незмінною структурою даних є кортеж.
Кожне окреме значення кортежу, яке може бути будь-якого типу, називають елементом кортежу. Синтаксис запису: круглі дужки та елементи через кому.

programming_languages = ('Python', 'C++', 'Javascript', 'Go', 'Java')

При однакових можливостях щодо використання, кортежі мають деякі особливості:

  • незмінність кортежу забезпечує захист від випадкових змін даних;
  • займають менше об'єму пам'яті при однакових розмірах, порівняно зі списком;
  • більш висока швидкодія виконання операцій над елементами такого типу послідовності;
  • можливість використання у якості ключа у структурі даних словник.
    Доступ до елементів кортежу є аналогічним як і у списку.
    Для перетворення списку у кортеж використовують функцію tuple()
programming_languages = ['Python', 'C++', 'Javascript']
tuple_value = tuple(programming_languages)

2. Словники

Теорія - 30 хв, практика - 0 хв

Словник - змінювана структура даних, яка є послідовністю із пар елементів "ключ:значення", записаних через кому у фігурних дужках.

personal_info = {'name': 'Nazariy',  
                  'birth_day': '2000-07-24', 
                  'country': 'Ukraine'}

Ключі в межах словника мають бути унікальними, тобто двох однакових ключів в словнику бути не може. Ключ повинен мати незмінюваний тип даних: ціле або дійсне число, рядок, кортеж. Враховуючи зазначену структуру, словник інколи ще називають асоціативним масивом.
Для оголошення порожнього словника може бути використано два варіанти синтаксису:

# Варіант 1
empty_list = {}
# Варіант 2
empty_list = dict()

Для оголошення словника з елементами є також два варіанти синтаксису:

# Варіант 1
empty_list = {'name': 'Nazariy', 
              'birth_day': '2000-07-24', 
              'country': 'Ukraine'}
# Варіант 2
empty_list = dict(name='Nazariy', 
                  birth_day='2000-07-24', 
                  country='Ukraine')
Важливо зазначити, якщо використовувати у якості ключа кортеж або число - створити такий словник можливо лише за першим варіантом синтаксису.

Доступ до елементів словника здійснюється за допомогою ключів, які подібно до індексів у списку / кортежу записуються у квадратні дужки.

personal_info = {'name': 'Nazariy', 
                'birth_day': '2000-07-24', 
                'country': 'Ukraine'}
personal_info['name']
# 'Nazariy'
personal_info['birth_day']
# '2000-07-24'

2.1 Редагування значень словника

Відповідно редагування значень словника можливо застосувавши ключ

personal_info = {'name': 'Nazariy', 
                'birth_day': '2000-07-24', 
                'country': 'Ukraine'}
# Зміна елементу
personal_info['name'] = 'Bogdan'
# {'name': 'Bogdan', 'birth_day': '2000-07-24', 'country': 'Ukraine'}
# Додавання елементу
personal_info['age'] = 24
# {'name': 'Bogdan', 'birth_day': '2000-07-24', 'country': 'Ukraine', 'age': 24}
# Видалення елементу
del personal_info['age']
# {'name': 'Bogdan', 'birth_day': '2000-07-24', 'country': 'Ukraine'}
Для додавання у словник нової пари 'ключ:значення' необхідно в одній дії визначитися та записати у квадратні дужки ключ і через оператор присвоювання значення для цього ключа

2.2 Методи словника

dict.fromkeys(iterable [, value]) - Створює новий словник, ключами якого будуть елементи з iterable і однаковим для всіх значенням value. 

>>> d.fromkeys(['a', 'b', 'c'],12) 
{ 'a': 12, 'b': 12, 'c': 12}}

dict.update([other]) - Доповнює словник dict парами (ключ=значення) з словника other,якщо ключ вже присутній в словнику, то його значення оновлюється. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> s={'A1':13, 'A5':15} 
>>> d.update(s) 
>>> print(d) 
{'A1': 13, 'A2': 3, 'A3': 5, 'A4': 7, 'A5': 15}

dict.copy() - Повертає копію словника dict

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> e=d.copy() 
>>> print(e) 
{'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7}

dict.get(key[, default]) - Повертає значення зі словника dict за ключем key. У випадку відсутності елемента з ключем key
повертається значення default. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.get('A2', 10) 3 
>>> d.get('A8', 10) 10

dict.setdefault(key[, default]) - Повертає значення зі словника dict за ключем key. У випадку відсутності елемента з ключем key повертається значення default, і до словника додається елемент з ключем key і значенням default. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.setdefault('A1', 13) 
1 
>>> d.setdefault('A5', 13) 
13 
>>> print(d) 
{'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7, 'A5': 13}

dict.keys() - Повертає ключі елементів словника dict у вигляді об'єкту перегляду словника, що забезпечують динамічний перегляд записів
словника.

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.keys() 
dict_keys(['A1', 'A2', 'A3', 'A4'])

dict.values() - Повертає значення елементів словника dict у вигляді об'єкту перегляду словника, що забезпечують динамічний перегляд записів словника. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.values() dict_values([1, 3, 5, 7])

dict.items() - Повертає ключі та значення елементів словника dict у вигляді об'єкту перегляду словника, що забезпечують динамічний
перегляд записів словника. Елементи словника подаються в вигляді кортежів
(ключ, значення). 

>>> d={'A1': 1, 'A2': 3, 'A3': 5} 
>>> d.items() 
dict_items([('A1', 1), ('A2', 3), ('A3', 5)])

dict.pop(key[, default]) - Вилучає зі словника dict елемент з ключем key та повертає його значення, як результат виконання функції. У випадку відсутності елемента з ключем key повертається значення default. Якщо default не вказаний, і елемент з ключем key відсутній, то виникає виняток KeyError. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.pop('A2') 
3 
>>> print(d) 
{ 'A1': 1, 'A3': 5, 'A4': 7}

dict.popitem() - Вилучає і повертає пару (ключ, значення) зі словника dict, як результат виконання функції. Пари повертаються в порядку LIFO (last-in first-out). Якщо словник порожній, то виникає виняток KeyError. 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> print(d) 
{'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.popitem() 
('A4', 7) 
>>> print(d) 
{'A1': 1, 'A2': 3, 'A3': 5}

dict.clear() - Очищує словник dict (вилучає всі елементи зі словнику). 

>>> d={'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> print(d) 
{'A1': 1, 'A2': 3, 'A3': 5, 'A4': 7} 
>>> d.clear() 
>>> print(d)
{}

3. Множини

Теорія - 20 хв, практика - 0 хв

Множина - це структура даних, що містить невпорядкований набір унікальних елементів. Множини в мові Python досить подібні до множин у математиці. Використання множин є доцільним у тому випадку, коли присутність елемента в наборі важливіша порядку слідування елементів та кількості їх повторень. Множина може містити елементи різних типів, проте ці елементи можуть бути лише незмінюваних типів даних: числа, рядки, кортежі. Вимога незмінності елементів множини накладається особливостями подання множини в пам'яті комп'ютера. Сама множина є змінюваним типом даних, тому до множин можна додавати нові та видаляти елементи, які вже існують. Як і у випадку математичних множин, у мові Python передбачено виконання операцій над множинами:

  • об'єднання,
  • перетину,
  • різниці,
  • симетричної різниці.

На відміну від списків, де елементи зберігаються у вигляді впорядкованої послідовності, у множинах порядок зберігання елементів невизначений. Це дозволяє виконувати операції типу "перевірити приналежність елемента множині" швидше, ніж просто перебираючи всі елементи множини.
Множини записуються, як перелік елементів, розділених комою та взятих у фігурні дужки: {1, 2, 3, 'Hello'}.

3.1. Задання множини

Задання порожньої множини виконується з використанням функції set(): 

>>> a = set()

Використання «{}» призведе до створення порожнього словника: 

>>> s={} 
>>> type(s) 
<class 'dict'>

Проте можна задати множину, перерахувавши її елементи, взяті в фігурні дужки «{}»: 

>>> s1 = {1, 2, 3} 
>>> s2 = {'a', 'b', 'c', 5}

Множину можна отримати з елементів об'єкта, яка є ітерабельною (діапазон, список, рядок, словник, кортеж, файл і т.д.), використавши функцію set([iterable]). Проте варто пам'ятати, що до множини будуть включені лише унікальні елементи. 

>>> a = set('hello') 
>>> a 
{'e', 'o', 'l', 'h'} 
>>> set(range(5))
{0, 1, 2, 3, 4}

3.3. Порівняння множин

Множини можна порівнювати між собою. Порівняння множин зводиться до перевірки, чи є множини рівними, або чи є одна множина підмножиною іншої. Нижче приведені множини, які будуть використовуватися у прикладах операцій порівняння:

>>> a={1,2,3} 
>>> b={1,2,3} 
>>> d={1,2,3,4} 
>>> c={4,5,6}

set == other - Перевірка, чи множини set та other рівні. Повертає True, якщо всі елементи множини set належать множині other, і всі елементи множини other належать множині set, інакше – False. 

>>> a == b 
True 
>>> a == d 
False

set != other - Перевірка, чи є множини set та other не рівними. Повертає True, якщо принаймні один елемент множини set не належать множині other, або принаймні один елемент множини other не належать множині set, інакше – False. 

>>> a != b 
False 
>>> a != d 
True

set <= other - Перевірка, чи є множина set підмножиною множини other. Повертає True, якщо всі елементи множини set належать множині other, інакше – False. 

>>> a<=d 
True 
>>> a<=b 
True

set < other - Повертає True, якщо всі елементи множини set належать множині other, але не всі елементи множини other належать множині set, інакше – False. 

>>> a<d 
True 
>>> a<b 
False

set.isdisjoint(other) - Повертає True, якщо множини set і other не мають спільних елементів, інакше – False. 

>>> a.isdisjoint(c) 
True 
>>> c.isdisjoint(d) 
False

set.issubset(other) - Перевірка, чи є множина set підмножиною множини other. Аналогічно до set <= other.

set.issuperset(other) - Перевірка чи є множина other підмножиною множини set. Аналогічно до set >= other.

3.4. Методи множин

set.add(x) - Додає елемент x до множини set. 

>>> s = {1, 2, 3} 
>>> s.add(4) 
>>> print(s) 
{1, 2, 3, 4}

set.remove(x) - Вилучає елемент x із множини set. Якщо такого елемента в множині немає, то виникає виняток KeyError.

set.discard(x) - Вилучає елемент x із множини set. Якщо такого елемента в множині немає, то нічого не відбувається.

set.pop() - Вилучає «перший» елемент з множини set та повертає його значення, як результат виконання функції. Так як множина – це невпорядкований набір, то не можна бути впевненим який з елементів буде взятий як перший. Якщо множина порожня, то виникає виняток KeyError.

set.clear() - Очищує множину set (вилучає всі елементи з множини).

3.5 Операції з множинами

Нижче приведені множини, які будуть використовуватися у прикладах операцій:

>>> d={1,2,3,4} 
>>> c={4,5,6}

set.union(*other) або set | other | .... - Повертає об’єднання множин set і other.Множина-результат буде містити як елементи множини set, так і елементи множини other. 

>>> d | c 
{1, 2, 3, 4, 5, 6}

set.intersection(*other) або set & other & ....
Повертає перетин множин set і other. Множина-результат буде містити елементи, які належать як множині set, так і множині other. 

>>> d & c 
{4}

set.difference(*other) або set - other - .... - Повертає
різницю множин set і other. Множина-результат буде містити елементи множини set, які не належать множині other. 

>>> d - c 
{1, 2, 3} 
>>> c - d 
{5, 6}

set.symmetric_difference(*other) або set ^ other - Повертає симетричну різницю множин set і other. Множина-результат буде містити елементи, які належать множинам set та other, але не належать обом множинам. Аналогічно до (set | other)-( set & other). 

>>> d^c 
{1, 2, 3, 5, 6}

Як можна бачити, вказані методи не призводять до зміни множини, а за ними відбувається формування нової множини.
Проте для множин передбачені методи, що будуть призводити до зміни початкової множини.

set.update(*other) або set |= other - Додає до множини set всі елементи множини other. Множина set буде містити об’єднання множин set і other.

set.intersection_update(*other) або set &= other - Вилучає з множини set всі елементи, які не входять до множини other. Множина set буде містити перетин множин set і other.

set.difference_update(*other) або set -= other - Вилучає з множини set всі елементи, які входять до множини other. Множина set буде містити різницю множин set і other.

set.symmetric_difference_update(other) або set ^= other - Множина set буде містити симетричну різницю множин set і other.

Висновки:

  • структури даних: список, кортеж, словник, множина, дозволяють впорядкувати дані для їх подальшої обробки та збереження;
  • зазначені структури даних мають свої методи, які дозволяють здійснювати операції редагування, сортування та пошуку визначених елементів.
